Key features of
MPanel-Pro

  • Powerful model building, form finding and patterning with a simple user interface
  • Time saving library of parametric models built in.
  • Conversion entire 3D mesh model to fabric panels with single button
  • Compensation all panels with single button
  • Seam and hem all panels with single button
  • Output to multiple formats for plotter/cutter or hand cutting.

Work Flow

1
Define boundary fixing points
2
Add mesh(es)
3
Form finding (relaxation)
4
Convert mesh(es) to panels
5
Compensate panels
6
Add seams and hems (multiple panels simultaneously)
7
Stamp panel numbers and details
8
Output to file(for plotting), XY coordinates, skeleton
