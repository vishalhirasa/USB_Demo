Key Features of
MPanel FEA

  • Easy to understand user interface makes these complex analysis tools accessible to most users
  • Powerful non-linear solver is unmatched in its abilities to handle complex conditions such as slack fabric
  • Integrated calculators to help determine material properties and loads
  • Time saving and simple database allows users to add fabrics, cables, poles
  • Results provided in both graphical models as well as raw data for use in Excel or importing into other structural analysis programs.
